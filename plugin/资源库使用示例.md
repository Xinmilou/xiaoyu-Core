# 向日葵资源库使用示例

## 1. 图片资源

### 1.1 随机图片
**命令**：`#随机图片`
**功能**：获取一张随机图片
**示例**：
```
用户：#随机图片
机器人：芝士你要的图片 [图片]
```

## 3. B站视频点播

### 3.1 使用BV号
**命令**：`#B站视频 BV1xx411c7m9` 或 `#播放B站视频 BV1xx411c7m9`
**功能**：获取指定BV号的B站视频
**示例**：
```
用户：#B站视频 BV1xx411c7m9
机器人：[视频预览或封面图] 这是你点播的视频喵(●ˇ∀ˇ●)
```

### 3.2 使用AV号

**示例**：
```
用户：#B站视频 av123456
机器人：[视频预览或封面图] 这是你点播的视频喵(●ˇ∀ˇ●)

### 3.3 使用b23.tv短链接

**示例**：

```

## 4. 注意事项

1. 所有命令支持带`#`或不带`#`前缀
2. 部分命令支持`随机`前缀，效果与不带前缀相同
3. 视频命令支持带`视频`后缀或不带`视频`后缀
4. B站视频命令支持`B站视频`或`播放B站视频`前缀
5. 资源分享功能可以通过修改`DEFAULT_SHARING`常量来启用或禁用
6. 如果资源获取失败，机器人会返回相应的错误提示

## 5. 命令列表速查表

| 功能类型 | 命令格式 | 功能描述 |
|---------|---------|---------|
| 图片 | `#随机图片` | 获取随机图片 |
| 图片 | `#(随机)?东方图` | 获取东方图片 |
| 图片 | `#(随机)?二次元图` | 获取二次元图片 |
| 图片 | `#妹子图` | 获取妹子图片 |
| 图片 | `#ks网红` | 获取KS网红图片 |
| 图片 | `#cos图` | 获取COS图片 |
| 图片 | `#(随机)?waifu` | 获取waifu图片 |
| 图片 | `#(多张|批量)?waifu` | 获取多张waifu图片 |
| 图片 | `#waifu搜索\s*` | 根据标签搜索waifu图片 |
| 图片 | `#waifu收藏|^#?我的收藏` | 获取waifu收藏列表 |
| 视频 | `#白丝(视频)?` | 获取白丝视频 |
| 视频 | `#黑丝(视频)?` | 获取黑丝视频 |
| 视频 | `#jk(视频)?` | 获取JK视频 |
| 视频 | `#高质量小姐姐(视频)?` | 获取高质量小姐姐视频 |
| 视频 | `#热舞(视频)?` | 获取热舞视频 |
| 视频 | `#萝莉(视频)?` | 获取萝莉视频 |
| 视频 | `#小姐姐(视频)?` | 获取随机小姐姐视频 |
| B站视频 | `#(播放)?B站(视频)?[:：]?\s*(.+)$` | 获取指定B站视频 |
